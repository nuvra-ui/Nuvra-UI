# Tooltip

A tooltip is a small pop-up box that appears when a user hovers over or focuses on an element, providing additional information.

## Setup

<Steps>

### Import the component

```tsx
import { Tooltip } from "@nuvra-ui/react";
```

### Customize the component Style

```tsx filename="TooltipDemo.tsx"
import { Tooltip } from "@nuvra-ui/react";

export function TooltipDemo() {
  return (
    <div className="flex gap-10">
      <Tooltip content="Click me" placement="left">
        <Link>Left</Link>
      </Tooltip>
      <Tooltip content="Click me" placement="right">
        <Link>Right</Link>
      </Tooltip>
      <Tooltip content="Click me" placement="bottom">
        <Link>Bottom</Link>
      </Tooltip>
      <Tooltip content="Click me" placement="top">
        <Link>Top</Link>
      </Tooltip>
    </div>
  );
}
```

# Contributing

Contributions are welcome! Here's how to get started.


## How to Contribute

1. **Fork & Branch**: Fork the [repository](https://github.com/nuvra-ui) and create a branch for your feature.
2. **Development**: Implement your changes.
3. **Test & Lint**: Run the linting suite to ensure code quality.
4. **PR**: Submit a pull request with a brief description of your changes.

```bash
# Clone your fork
git clone https://github.com/nuvra-ui/nuvra-ui
cd nuvra-ui

# Install dependencies
pnpm install

# Start the development environment
turbo dev
```

## Guidelines

* **Commits**:  Write clear commit messages and folow the [`.commitlintrc.json`](https://github.com/nuvra-ui/Nuvra-UI/blob/main/.commitlintrc.json) rules.
* **TailwindCSS**: All styling must be done via TailwindCSS utility classes.
* **Documentation**: Please include a short example for any new component or feature.
